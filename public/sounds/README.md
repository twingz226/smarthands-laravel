# Notification Sounds

This directory contains notification sound files for the application.

## Adding a Notification Sound

1. Place your notification sound file (in MP3 format) in this directory.
2. The default sound file should be named `notification.mp3`.
3. The sound should be short (1-2 seconds) and not too loud or annoying.

## Recommended Free Sound Sources

- [FreeSound](https://freesound.org/)
- [ZapSplat](https://www.zapsplat.com/)
- [SoundBible](https://soundbible.com/)

## File Format

- Format: MP3
- Sample Rate: 44.1 kHz
- Bitrate: 128 kbps or higher
